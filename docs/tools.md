# Tools

## Summary Generator

Produces a concise Markdown summary from full document text.

## Outline Generator

Generates a hierarchical outline based on the summary output.

## Insights

Produces a structured report with key themes, quotes, and contradictions.

## Infographic

Creates a visual infographic using the OpenRouter image-capable model.

## PDF Generator

Renders one image per 5 chunks and compiles them into an A4 PDF.
