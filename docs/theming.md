# Theming

OpenRecords uses CSS variables in `frontend/src/styles/globals.css`.

## Themes

- Bi Dark
- Bi Light
- Matcha
- Mocha
- Nord
- Gruvbox

## How It Works

- Theme is applied via `data-theme` on the `html` element.
- The theme toggle stores the choice in localStorage.
- CSS variables drive all colors and typography.
